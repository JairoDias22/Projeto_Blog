Mini Blog em PHP

Este projeto é um mini blog desenvolvido como atividade do curso Técnico em Informática para Internet do SENAC. O objetivo é criar uma estrutura simples de blog utilizando PHP, HTML, CSS e Bootstrap, exibindo posts de forma dinâmica a partir de arrays associativos.

O site possui uma página inicial onde os posts são listados com imagem, título e resumo. Ao clicar em um post, o usuário é levado para uma página individual que exibe o conteúdo completo, além da categoria e das tags relacionadas. O layout é responsivo e utiliza componentes básicos do Bootstrap.

O tema escolhido para os posts foi o Google Gemini, destacando o que é o modelo, suas aplicações e comparações com outras inteligências artificiais.
